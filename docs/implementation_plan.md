# 🌌 實作計畫：時空與意識互動模擬平台 (Track 3)
`Interactive Spacetime & Consciousness Simulator`

本計畫旨在實作一個兼具前沿科技感與哲學美感的 Web 互動應用程式。透過直觀的視覺化與互動機制，將《時間並非真實存在》研究計畫中極度抽象的物理與形上學理論（塊狀宇宙、大腦意識調頻、ER=EPR 非定域性）具體呈現。

---

## 🛠️ 技術堆疊 (Tech Stack)

為了實現極致的視覺美感、流暢的微動畫與高效的 3D 渲染，我們建議使用以下技術：
*   **建置工具**：Vite (提供極速的開發與熱更新體驗)
*   **前端框架**：React + TypeScript (結構化組件開發，便於維護狀態)
*   **視覺與 3D 渲染**：Three.js / React Three Fiber (用於塊狀宇宙 3D 膠卷與蟲洞拓撲渲染)
*   **動畫庫**：Framer Motion (用於平滑的 UI 微動畫與過渡效果)
*   **樣式與美學**：Vanilla CSS (客製化 CSS 變數、毛玻璃效果/Glassmorphism、暗色系科幻漸層)

## 🎯 核心功能模組

### 1. 🎬 塊狀宇宙 (Block Universe) 互動膠卷
*   **概念**：時間並非流動，而是像沖洗好的底片一樣同時存在。
*   **呈現**：一個 3D 旋轉的時間膠卷（或者 2D 平滑卷軸）。
*   **互動**：使用者可以拖曳時間軸（過去 $\rightarrow$ 現在 $\rightarrow$ 未來），膠卷中的「時空切片 (Slices)」會隨之放大，並動態投影出該時間點的宇宙事件變化（例如：恆星由星雲塌縮、發光到最後超新星爆炸的過程）。

### 2. 🧠 大腦微管 (Orch-OR) 意識調頻旋鈕
*   **概念**：大腦是量子接收器，意識的心念頻率決定了我們將疊加態塌縮成哪一個版本的現實。
*   **呈現**：一個高科技質感的復古收音機旋鈕（調頻控制台），配有動態示波器（Oscilloscope）。
*   **互動**：當使用者旋轉旋鈕改變「意識定錨頻率」時，相同的時空切片會呈現出完全不同的現實投影：
    *   *頻率 A（物質流）*：呈現傳統的物理實體世界。
    *   *頻率 B（資訊流）*：物理實體淡出，轉為純粹的量子代碼與糾纏線條。
    *   *頻率 C（意識流）*：世界轉為溫慢的哲學光斑與心靈金句。

### 3. 🌀 ER = EPR 量子非定域性橋樑
*   **概念**：宇宙底層是一個沒有空間的「單一個點」，糾纏粒子透過蟲洞相連。
*   **呈現**：一個互動拓撲圖。畫面上散佈著多個看似相距極遠的粒子。
*   **互動**：點擊粒子時，時空背景會像紙張一樣「折疊」，讓兩顆粒子在底層重合為同一個點，並拉出一條代表愛因斯坦-羅森橋（蟲洞）的立體光帶。

### 4. 🕯️ 永恆主義心靈金句生成器
*   **概念**：過去不曾消失，宇宙不會遺忘。
*   **呈現**：隨著使用者在時間軸穿梭或調整頻率，畫面上會以精美的排版動態浮現具備詩意與療癒感的存在主義金句。

## 📋 預計實作步驟 (Implementation Phases)

### 📌 Phase 1: 專案初始化與設計系統 (Design System)
1. 使用 Vite 初始化 React + TypeScript 專案。
2. 配置基本的設計系統（以深邃太空黑為主色調，搭配量子藍、意識紫的霓虹漸層與毛玻璃面板樣式）。
3. 建立基礎佈局與導覽框架。

### 📌 Phase 2: 塊狀宇宙 3D 膠卷開發
1. 實作基於 React Three Fiber (Three.js) 的 3D 膠卷模型。
2. 串接時間 Slider 控制器，實現時空切片隨時間軸平滑切換的動畫。
3. 繪製並設計時空切片內部的視覺內容（利用 Canvas/WebGL 繪製演化星體或幾何圖形）。

### 📌 Phase 3: 意識調頻旋鈕與多現實塌縮
1. 設計並實作 SVG/Canvas 基礎的圓形旋鈕組件，支援滑鼠/觸控拖曳旋轉。
2. 建立「意識頻率狀態機」，當頻率改變時，發送事件切換 3D 場景的渲染模式（物理態、資訊態、意識態）。
3. 實作波形示波器，隨頻率調整呈現相應的波形震盪。

### 📌 Phase 4: ER=EPR 空間折疊互動
1. 實作粒子網格，滑鼠懸停時會產生微弱的量子糾纏漣漪。
2. 設計「空間折疊」的著色器（Shader）或 CSS 3D 轉換動畫，展現空間維度被拉平、粒子合一的震撼視覺。

### 📌 Phase 5: 精緻化與體驗優化
1. 加入平滑的入退場 View Transitions 動態效果。
2. 加入背景氛圍音效（可選，如低頻宇宙微波背景輻射噪聲，提供沈浸感）。
3. 自動適應行動端與桌面端版面。

## 🔬 驗證計劃 (Verification Plan)

### 自動化建置與檢查
*   執行 `npm run build` 確認 TypeScript 與生產打包無任何編譯錯誤。
*   使用 ESLint 進行靜態程式碼檢查，確保無語法隱患。

### 手動驗證與 UX 測試
*   **流暢度測試**：確認 3D 膠卷在 60 FPS 下平滑運行，無卡頓。
*   **調頻交互**：旋轉旋鈕時，場景切換的過渡動畫是否足夠自然、具備「塌縮」的儀式感。
*   **響應式佈局**：在 Chrome DevTools 中切換為 iPhone/iPad 視角，確認 UI 佈局正常。
